# Tech Stack: Chatinho

## Frontend & Backend
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS

## Database & ORM
- **Database**: PostgreSQL (via Docker)
- **ORM**: Prisma

## Real-time & Integration
- **Real-time**: Socket.io (Separate server planned for Phase 2)
- **WhatsApp API**: Evolution API (via Docker)
- **AI**: OpenAI API

## Authentication
- **Provider**: NextAuth.js (Credentials)

## Infrastructure
- **Containerization**: Docker & Docker Compose
- **Deployment**: VPS Linux (Planned)
